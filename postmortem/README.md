## 🚨 关键模式识别

代码审查时，警惕以下模式：

### 安全相关

```javascript
// 🚨 XSS 风险
element.innerHTML = userInput;
```

```c
// 🚨 内存安全
free(ptr);
// ptr 未置空
```

### 并发相关

```python
# 🚨 无锁共享状态
self.shared_dict[key] = value
```

### 资源管理相关

```python
# 🚨 无清理逻辑
try:
    resource = acquire()
except:
    pass  # 资源未释放
```

### 测试相关

```python
# 🚨 固定延迟
await asyncio.sleep(2)  # 可能超时
```
